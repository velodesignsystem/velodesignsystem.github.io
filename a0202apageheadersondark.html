Page header component
The header also includes navigation actions the user can take on the left and right side, indicated by the icons. The left side should always be reserved for the back nav arrow, applied if needed. Options on the right side can vary; common usages include 3 dots menu, share icon, trash can for delete and X icon for close, applied if needed.
Page headers can be found on every screen of the app. There are variations between the iOS and Android version, but only due to the difference in system status bar.

There are no scenarios where a two line title is required in an On-dark background, we will add it if it become necessary.

The header title will always be in title case, as shown above, with each word begining with a capital letter.

There are variations where there are attachments at the bottom of the header object, as shown above with the page title attachment, the search bar attachment, and the carousel attachment.

Header application
The On-Dark header has a range of spacing variations for closest objects, depending on available space and visual balance.
For Velo logo pages, the Velo logo is prominantly displayed on the sign up and log in screens, and there is a need to display keyboard woithout blocking any CTAs or links. Here the header component does not have title and appears empty, the logo moves into the header area -12px.

Transaction details page icons begins at 0px from the header object. The close proxiity of the icon to the Page header title helpts to give the user context as to what kind of trnasaction they are reviewing, and smoothly flows down into the transaction value and card details.

Message pages are a conversational, light weight interaction with users. Here the page header title is secondary to the large conversational text below, it works to have the text 12px away from the header object, giving its own space and presence.

For icon confrimation pages, image pages, 18px from header object clears some room for the image to become the hero of the page. List pages require some distance from the header title to not feel too cluttered, 18px works here too.
